Storage transparency

Cookies and local storage

Draft version: 23 July 2026

Draft for legal review. The technical state described below is intentionally designed not to require a consent banner. The website provider is Healthcare Data Solutions s. r. o.; DevelopED is its commercial brand.

Current state: this website sets no cookies and uses neither localStorage nor sessionStorage. It has no analytics, advertising, embedded player, chat widget or contact form. A cookie banner is therefore not displayed for this configuration.

What the website uses

  • static HTML, CSS and a small JavaScript file served from developed.sk;
  • technical hosting logs for security and diagnostics, which are server-side records rather than cookies or browser storage;
  • links to email, telephone, LinkedIn and presented projects; an external service loads only after you follow its link.

Website language

Slovak and English have separate URLs, so the language choice is not stored in the browser.

Future changes

If analytics, marketing, embedded content or another non-essential technology that stores or reads information on a device is added, it will first undergo technical and legal assessment. Anything requiring consent will stay blocked until active consent; refusal will be as easy as acceptance and the decision can be changed later.

How to verify the state

You can inspect Cookies, Local Storage and Session Storage in browser developer tools. Network requests on initial page load should go only to developed.sk.

Contact

Send technology questions to info@developed.sk. Related personal-data processing is explained in the privacy notice.
