Frequently asked questions

Where is assistant data processed?

It depends on the configuration. The server can be located in the EU, while messaging, model APIs, search and other integrations must be assessed separately. These flows are documented before launch.

Do model providers use data for training?

Terms differ by provider, product, plan and settings. The solution design therefore identifies approved services, accounts, retention rules and available contractual safeguards.

Do users need technical knowledge?

Day-to-day use can happen through a familiar messaging channel. The client still appoints someone responsible for access, integration approval and security decisions.

What is included in support?

The quotation or service agreement defines its exact scope, response targets, updates and limitations.
