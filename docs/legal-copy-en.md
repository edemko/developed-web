# 3. Cookies and device storage

> This section is accurate only after removal of the language-preference `localStorage` and a clean production audit.

## Current use

In its target configuration, the website:

- does not set cookies;
- does not use `localStorage`, `sessionStorage`, IndexedDB or similar device storage;
- does not use analytics, advertising or marketing identifiers;
- does not load embedded maps, videos, chat widgets or social pixels.

Language is selected through a normal link between the Slovak URL `/` and the English URL `/en/`; the choice is not stored on the device.

For that reason, the site does not display a consent banner. A banner would serve no purpose when the website uses no technology requiring consent.

## Server logs

Hosting and security infrastructure may process IP addresses and technical request data in server logs. A server log is not a cookie or device-storage technology, but it is personal-data processing. It is described in the privacy notice.

## Third-party links

When a visitor opens a project, LinkedIn or another external link, that third party's rules apply. DevelopED does not set the third party's cookies; the visitor has left our website.

## Future changes

Before introducing analytics, advertising pixels, chat, maps, videos, A/B testing or other external content, we will conduct a technical and legal review.

If a technology is introduced that is not strictly necessary for a service expressly requested by the user:

- it will not load before valid consent;
- rejecting it will be as easy as accepting it;
- users will be able to choose individual purposes;
- consent can be withdrawn at any time;
- this page will list the exact providers, purposes and durations.

Cookie and storage questions: [info@developed.sk](mailto:info@developed.sk).
